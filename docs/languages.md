# Language Options

<a id="english"></a>

## English

Agent Extension Auditor helps you review local AI agent extensions before you trust them. It scans skills, plugins, MCP servers, hooks, configs, and package metadata, then groups findings by extension type and severity. Everything stays local: no telemetry, no account, no upload, and no secret value printing.

```bash
npm install -g agent-extension-auditor
agent-audit scan
agent-audit ui
agent-audit scan --format html --output risk-report.html
```
